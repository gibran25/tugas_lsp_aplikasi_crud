# Aplikasi Pendataan Buku Tamu

<li>Aplikasi crud berbasis web yang dibuat menggunakan framework php Codeigniter dan css Bootstrap.</li>
<li>Aplikasi berfungsi untuk melakukan crud pada data buku.</li>
